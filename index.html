Colliding Blocks Compute π

Perfectly elastic collisions between two 1D blocks and a wall. The total number of collisions approximates digits of π when the large mass is a power of 100.

Collisions: 0
Status: Ready

How it works

  • Setup: The large block starts at rest near the wall. The small block starts to the right, moving left.
  • Collisions: All collisions are perfectly elastic. We count block-block and block-wall collisions.
  • Stop: The simulation ends when both blocks move right and can no longer collide.
  • π connection: Set large mass to 100 for 2 digits (31), 10,000 for 3 digits (314), etc.

Try: small mass = 1, large mass = 100 → 31 collisions. Large mass = 10,000 → 314 collisions.
